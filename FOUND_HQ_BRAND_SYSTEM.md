# Found HQ Brand System

## Principle

Found HQ is a private operating instrument, not a generic SaaS dashboard. It should feel calm, exact, and unmistakably Found.

## Palette

- Found Black: `#080A09`
- Found White: `#F7F8F7`
- Signal Green: `#32D074`
- Quiet text: `rgba(247,248,247,0.48)`
- Hairline: `rgba(255,255,255,0.08)`
- Attention: `#D9BD68`

Signal Green is reserved for identity, selection, and decisive actions. It is not used as repeated furniture.

## Typography

- Native Apple system stack.
- Page title: 34px mobile, 36px desktop, restrained semibold.
- Section title: 17px mobile, 13px desktop, semibold.
- Operational row title: 17-18px mobile, 14-17px desktop, semibold.
- Metadata: 13-15px mobile, 10-11px desktop, regular.
- Letter spacing remains zero.

## Shape

- Operational lists use hairline separators, not cards.
- Controls use 4px corners.
- Status is typographic, not pill-shaped.
- Rounded pills are not part of the Found HQ language.

## Hierarchy

1. Current page and purpose.
2. Work requiring a decision.
3. Relationship or prospect identity.
4. Supporting metadata.
5. Secondary controls.

## Interaction

- Repeated row actions are quiet.
- White is the default command color.
- Green appears when selection or commitment matters.
- Mobile controls must remain usable with one hand and never require horizontal scrolling.
